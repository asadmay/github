Быстрый старт GitHub Pages

Вы можете использовать GitHub Pages, чтобы продемонстрировать некоторые проекты с открытым исходным кодом, разместить блог или даже поделиться своим резюме. Это руководство поможет вам приступить к созданию вашего следующего веб-сайта.

Вступление

GitHub Pages— это общедоступные веб-страницы, размещенные и опубликованные через GitHub. Самый быстрый способ приступить к работе — использовать средство выбора темы Jekyll для загрузки готовой темы. Затем вы можете изменить содержимое и стиль своих страниц GitHub.

Это руководство поможет вам создать пользовательский сайт на username.github.io.

Создание вашего веб-сайта

1. В правом верхнем углу любой страницы используйте раскрывающееся меню и выберите Новый репозиторий .

2. Введите username.github.io в качестве имени репозитория. Замените username своим именем пользователя GitHub. Например, если ваше имя пользователя octocat, имя репозитория должно быть octocat.github.io.

3. Под именем вашего репозитория нажмите Настройки (Settings).

4.В разделе Код и автоматизация (Code and automation) боковой панели нажмите Страницы (Pages) .

5.Щелкните Выбрать тему (Theme Chooser).

6. Откроется окно выбора темы. Просмотрите доступные темы, затем нажмите Выбрать тему (Selecte theme) , чтобы выбрать тему. Позже можно легко изменить тему, поэтому, если вы не уверены, просто выберите любую сейчас.

7. После того, как вы выберете тему, README.md файл вашего репозитория откроется в редакторе файлов. В README.md файле вы будете писать контент для своего сайта. Вы можете отредактировать файл или пока сохранить содержимое по умолчанию.

8. Когда вы закончите редактирование файла, нажмите Принять изменения.

9. Посетите username.github.io для просмотра вашего нового веб-сайта. Примечание. Публикация изменений на вашем сайте может занять до 20 минут после отправки изменений на GitHub.

Изменение заголовка и описания

По умолчанию название вашего сайта username.github.io. Вы можете изменить название, отредактировав _config.yml файл в своем репозитории. Вы также можете добавить описание для вашего сайта.

1. Перейдите на вкладку Код (Code) вашего репозитория.

2. В списке файлов нажмите, _config.yml чтобы открыть файл.

3. Нажмите , чтобы отредактировать файл.

4. Файл _config.yml уже содержит строку, определяющую тему для вашего сайта. Добавьте новую строку title:, за которой следует заголовок, который вы хотите. Добавьте новую строку description:, за которой следует описание, которое вы хотите. Например:

theme: jekyll-theme-minimal
title: Домашняя страница Octocat
description: Добавьте страницу в закладки, чтобы следить за обновлениями моего проекта!

5. Когда вы закончите редактирование файла, нажмите Принять изменения .
