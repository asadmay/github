GitHub Pages инструкции

Вы можете создать сайт прямо из репозитория на GitHub.com

GitHub Pages доступен в общедоступных репозиториях с GitHub Free и GitHub Free для организаций, а также в общедоступных и частных репозиториях с GitHub Pro, GitHub Team, GitHub Enterprise Cloud и GitHub Enterprise Server.

Быстрый старт для GitHub Pages

Вы можете использовать GitHub Pages, чтобы продемонстрировать некоторые проекты с открытым исходным кодом, разместить блог или даже поделиться своим резюме. Это руководство поможет вам приступить к созданию вашего следующего веб-сайта.

Начало работы с GitHub Pages

Вы можете настроить базовый сайт GitHub Pages для себя, своей организации или проекта.

Настройка сайта GitHub Pages с помощью Jekyll

Вы можете использовать популярный генератор статических сайтов Jekyll для дальнейшей настройки своего сайта GitHub Pages.
